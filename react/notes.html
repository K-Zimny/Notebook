Array State Management

When working with state where an array contains subarrays you need to map over each array at a time to access its parts.

Example

const INITIAL_GAME_STATE = [ [0, 0], [0, 0], [0, 0], [0, 0], ]; // INITIAL_GAME_STATE = [[0,0],[0,0],[0,0],[0,0]] setGameState((prevState) => { const newState = prevState.map((row, rowIndex) => rowIndex == r ? row.map((cell, cellIndex) => (cellIndex == c ? (cell = 1) : cell)) : row ); return newState; });

Data is stored in an array and that data is an array itself. Map over the first array to find the array you need to access. When you have the array you need to work with you then map over that array to work with its data.
